Типы:
1. Пустой
    * none
    * Занимает 2 ячейки
2. Ссылка
    * link
    * Занимает 2 ячейки
3. Команда
    * command
    * Занимает 2 ячеки (без аргументов)
4. Логическая величина
    * logic
    * Занимает 2 ячейки
5. Целое число
    * number
    * Занимает 2 ячейки
6. Строка
    * string
    * Занимает *длина(строки) + 2-3 ячейки*
7. Элемент словаря
    * dictionary item
    * Занимает 8 ячеек
